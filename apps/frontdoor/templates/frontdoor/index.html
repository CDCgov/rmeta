CDC Submission API (Frontdoor)

Hello, Intermediaries and STLT Partners!

Frontdoor is a new and straightforward way to submit data to the CDC. It is in alpha testing phase right now.

This site hosts RESTFul APIs for uploading data to CDC. It is based on the "CDC Submission Specification", which is a set of APIs and data formats that CDC uses to receive data from its partners. The basic premise of the API/Spec is that for a single item, two files are provided for submission. The first file is the "payload" file, which contains the actual data in HL7v2, FHIRBundle or CCDA format. The second file is the "metadata" file, which describes and identifies the submission. The metadata file is a JSON file that contains information about the payload file, such as its origins, type, and other relevant details. The API responds with an HTTP response containing a JSON body. The JSON body can be examined to see if the submission was "accepted" or "rejected" If rejected, the response contains the errors with the attempted submission. Failed submission can corrected and resubmitted. Details of the receipt file JSON format can be found in the "CDC Submission Specification".

Other Ways to Submit

Other methods available for submission:

  • Send data to an AWS S3 bucket in which your-organization has been granted access by CDC. Example: Upload files to: 
    your-org-id.partners.s3.partners.cdc.gov
  • Push data to an AWS S3 Bucket your organization controls but has been made accesiable to CDC. Example: Upload Files to: 
    cdcreports.s3.example.com
  • Send data to an SFTP folder in which your-organization has been granted access by CDC. Example: Upload Files to: 
    your-org-id@sftp.partners.cdc.gov
  • Push data to an SFTP folder that your organization controls but has been made accesiable to CDC. Example: Upload Files to: 
    cdcreports.sftp.example.com
  • Send email to CDC's Direct email service. Example: Attach files and send to: 
    your-org-id@direct.partners.cdc.gov
JSON Receipts will be delivered as a file on the S3/SFTP bucket/folder or as an email attachement via Direct email.

Please refer to the "CDC Submission Specification" for more information.
